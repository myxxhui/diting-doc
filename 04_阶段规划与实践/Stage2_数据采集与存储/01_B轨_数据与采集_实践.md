<a id="l4-stage2-b01-goal"></a>
## 步骤目标

- 按 [02_B轨数据与存储规约](../../03_原子目标与规约/B轨/02_B轨数据与存储规约.md) 落实 B 轨**专用**数据需求：**标的主营构成表**（symbol_business_profile）、**细分注册表**（segment_registry）；基本数据（行情/技术、标的池）由 11_ 与 A 轨采集提供，本步不重复实现。
- L2 存在 B 轨专用表：标的主营构成表、细分注册表；建表脚本或 Chart 可执行（表结构见 [12_右脑数据支撑与Segment规约](../../03_原子目标与规约/_共享规约/12_右脑数据支撑与Segment规约.md)）。
- 采集模块（或扩展）实现 **ingest_business_profile**（或等价任务），将标的主营构成写入 L2；细分注册表初始化/维护就绪；至少一条「数据源→存储→信号层/B 模块可读」路径可验收。

<a id="l4-stage2-b01-dor"></a>
## 准入（DoR）

- Stage2 采集基础设施与 L2 可写可读已就绪（见 [01_A轨_基础设施与依赖_实践](01_A轨_基础设施与依赖_实践.md)、[02_A轨_采集逻辑与Dockerfile_实践](02_A轨_采集逻辑与Dockerfile_实践.md)）。
- [02_B轨数据与存储规约](../../03_原子目标与规约/B轨/02_B轨数据与存储规约.md) 与 dna_b_track 的 `data_requirement`、`storage` 已对齐。

<a id="l4-stage2-b01-steps"></a>
## 实施步骤

**工作目录**：diting-core（或部署仓，若表由 Chart 创建）

1. **建表**：在 L2 创建 B 轨专用表（或通过 init 脚本/Helm Chart）：**标的主营构成表**（symbol_business_profile）、**细分注册表**（segment_registry）；表结构与 DDL 见 [12_右脑数据支撑与Segment规约](../../03_原子目标与规约/_共享规约/12_右脑数据支撑与Segment规约.md)。可选：细分信号缓存表（segment_signal_cache）由**信号层**写入，实现见 [06_B轨_信号层生产级数据采集_实践](06_B轨_信号层生产级数据采集_实践.md)；本步可仅建表或由 06 步建。
2. **采集扩展**：在现有采集管道中实现 **ingest_business_profile**（或合并入行业/营收采集）：从约定数据源拉取标的→主营业务/产品营收占比，映射为已注册 segment_id，写入标的主营构成表。**多层级（三层）**：同一标的**可写多行**，对应 **第 1 层 domain 级**（如 tech/agri/geo）、**第 2 层 板块级**（如 tech_ai）、**第 3 层 具体业务级**（如 tech_ai_factory）；见 [12_ §2.2 segment_id 分层约定](../../03_原子目标与规约/_共享规约/12_右脑数据支撑与Segment规约.md)。**可选**：按申万/行业标签为标的**补写**第 1 层、第 2 层 segment_id（如科技股自动挂 tech、tech_ai），使无具体业务时也能在信号层拉取「大盘+板块」信号；同一标的下营收占比最高的一行设 is_primary=True。细分注册表初始数据（含三层示例枚举）由脚本或采集前置步骤写入；若支持 tier/segment_tier 字段则与 12_ 扩展约定一致。
3. **消费验证**：信号层能按 symbol 查标的主营构成、按 segment_id 查细分注册表；B 模块能使用 L1 行情/技术数据（11_ 提供）与标的主营构成做阶段/趋势打分；无数据时降级策略符合 02_ 与 12_ 约定。

<a id="l4-stage2-b01-verify"></a>
## 验收

- [ ] B 轨专用表存在且 schema 与 02_、12_ 一致或兼容（symbol_business_profile、segment_registry）。
- [ ] ingest_business_profile（或等价步骤）能写入标的主营构成表，且 segment_id 均在细分注册表中；至少可按 symbol 查询到主营构成。
- [ ] L5 或本目录验收清单中「01_B轨 数据与采集」已勾选。

<a id="l4-stage2-b01-exit"></a>
## 准出

- 上述验收通过；设计/DNA 无变更或已同步。
