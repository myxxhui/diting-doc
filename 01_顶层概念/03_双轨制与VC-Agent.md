## 一、三轨制定义与资金比例

三轨定义、资金比例与协同价值见 [01_一句话定义与核心价值](./01_一句话定义与核心价值.md)#三轨制与分工。下文给出 TimeHorizon 语义、判官分流与分账/复盘分轨。

## 二、TimeHorizon 语义与使用场景

协议定义见 `diting-core/design/protocols/brain/expert.proto`（需扩展枚举）：

- **SHORT_TERM**：A 轨短线；看几天；判官施加 2% 硬止损、现金拖累、不可能三角约束。
- **MEDIUM_TERM**：B 轨中线；看 1～2 月；判官施加**中线止损**（如 5%～8% 或 ATR/结构）、**阶段止盈**规则，不施加 2% 硬止损与现金拖累。
- **LONG_TERM**：C 轨长线；看几年；判官豁免 2% 硬止损与现金拖累，仅施加逻辑证伪与大周期反转规则。

Router 或专家在输出 `ExpertOpinion` 时设置 `horizon`；判官（Module D）根据该字段分支执行。

## 三、判官对 MEDIUM_TERM（B 轨）的规则

当信号或专家意见携带 `TimeHorizon = MEDIUM_TERM` 时：

- **不施加**：2% 硬止损、现金拖累监控（与 A 轨隔离）。
- **施加**：**中线止损**（如 5%～8% 或 ATR/结构止损，可配置）、**阶段止盈**（如目标 30%/50%/100% 分级或阶段结束/景气拐点）；仓位与复盘按 B 轨分轨。

## 四、判官对 LONG_TERM（C 轨）的豁免规则

当信号或专家意见携带 `TimeHorizon = LONG_TERM` 时：

- **豁免**：不施加 2% 硬止损（单笔亏损 > 2% 不触发市价斩仓）；不施加现金拖累监控（Cash Drag Monitor 不作用于该轨仓位）。
- **仍施加**：
  - **逻辑证伪止损**：产业趋势或基本面逻辑被证伪时止损（例如：核心假设不再成立、政策/技术路线颠覆）。
  - **大周期反转止盈（Major Trend Reversal）**：识别到标的或行业进入长期下行周期时止盈/止仓（可执行定义见下）。

**大周期反转（Major Trend Reversal）可执行定义**（最小可落地版本）：
- **输入**：标的或行业维度的价格/均线/波动率等趋势指标。
- **判定主体**：VC-Agent 或判官内独立子模块；**C 轨专用**输入/输出与 schema 见 [03_/C轨/](../03_原子目标与规约/C轨/README.md)。
- **输出**：止盈/减仓信号。**逻辑证伪止损**：输入为事件与共识证伪条件，输出为平仓指令。

详见 [09_ 核心模块架构规约](../03_原子目标与规约/_共享规约/09_核心模块架构规约.md) Module D 三轨分流。

## 五、VC-Agent（C 轨长线）在 MoE 中的位置

- **位置**：Module C 专家池中的**信仰专家（VC-Agent）**；Router 根据标的或资金账户将请求路由至 VC-Agent，输出 `LONG_TERM`。
- **数据源**：基本面数据（财报、营收增速、研发占比等），与 A/B 轨技术面/阶段信号区分。
- **逻辑**：基本面驱动、买入锁仓、仅在大周期反转或逻辑证伪时平仓。详见 [05_C轨系统概念与目标](./05_C轨系统概念与目标.md)、[03_/C轨/](../03_原子目标与规约/C轨/README.md)。

## 六、A/B/C 轨分账与复盘分轨

- **分账**：资金与仓位按轨标识（A / B / C）分账管理。
- **B 轨资金**：比例可配置；**优先实现 B 轨**，再扩展 A、C。
- **复盘分轨**：A/B/C 三轨分开统计；B 轨统计 MaxDD、平均持仓周期、阶段止盈触发率、年化复利等。

## 七、三轨规范边界

以下规则按轨适用；01/02 中「仅 A 轨」「仅 B 轨」「仅 C 轨」以本表为准。

| 规则/模块/指标 | 仅 A 轨 | 仅 B 轨 | 仅 C 轨 | 共用 |
|----------------|--------|--------|--------|------|
| 2% 硬止损 | ✓ | — | — | — |
| 中线止损（5%～8% 或 ATR） | — | ✓ | — | — |
| 阶段止盈（30%/50%/100% 或阶段结束） | — | ✓ | — | — |
| 现金拖累监控 | ✓ | — | — | — |
| 不可能三角 KPI | ✓ | — | — | — |
| 逻辑证伪止损 | — | — | ✓ | — |
| 大周期反转止盈 | — | — | ✓ | — |
| 判官分流 | SHORT_TERM | MEDIUM_TERM | LONG_TERM | Module D 统一入口 |
| 账户/仓位标识 | A | B | C | 分账管理 |
| 复盘指标 | WinRate、CAGR、MaxDD | B 轨 MaxDD、持仓周期、止盈触发率、年化 | C 轨 MaxDD、持仓周期、逻辑证伪/反转 | 分轨统计 |
