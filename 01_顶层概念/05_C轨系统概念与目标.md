## 一、C 轨概念

**C 轨（长期捕手轨）** 是谛听三轨制中的**长期捕手系统**，致力于在未来 **3 年、5 年、10 年** 时间尺度上，**发现并持续持有具备高爆发成长潜力的公司**，目标回报为 **10 倍至百倍级别**。

| 维度 | 含义 |
|------|------|
| **发现** | 通过**严谨、深度的商业与基本面分析**，覆盖财报细节、管理层与激励、竞争格局、技术路线与研发、产业链与定价权等**所有关键商业信息**，形成可追溯的洞察与假设。 |
| **捕获** | 在满足 C 轨准入与共识的前提下建仓，经由 Router/VC-Agent 输出 `TimeHorizon = LONG_TERM`，判官走 C 轨分支（豁免 2% 硬止损与现金拖累）。 |
| **消化（拿得住）** | 依赖**逻辑证伪止损**与**大周期反转止盈**，而非短期波动；只在核心假设被推翻或大周期反转时减仓/止盈。 |

**与 A 轨、B 轨的区别**：A 轨短线现金流，B 轨中线阶段 30%～100%；C 轨追求**少数极高赔率标的的长期复利**，容忍途中回撤与长持有周期。

## 二、C 轨目标要求（超额完成标准）

| 维度 | 目标要求 | 超额完成含义 |
|------|----------|--------------|
| **时间视野** | 3 / 5 / 10 年 | 协议/配置支持多档；标的或共识可带时间视野标签。 |
| **回报目标** | 10×～100× | 策略/共识中显式包含「潜在空间」或「赔率」估计。 |
| **信息深度** | 全关键商业信息 | 采集、存储、消费覆盖细财报、管理层、竞争、研发、产业链与政策等。 |
| **分析严谨性** | 可追溯、可证伪 | 共识含结构化假设、证据摘要、证伪条件。 |
| **定力与消化** | 仅逻辑/大周期退出 | 逻辑证伪与大周期反转有明确输入、判定规则与执行接口。 |
| **与 A/B 轨隔离** | 资金、风控、复盘分轨 | 分账、仓位/订单轨标识、判官/风控分支、C 轨专用复盘指标均有规约。 |
