# 模型部署工程师（MLOps）

## 一、岗位概览

| 项目 | 内容 |
| --- | --- |
| 职位名称 | 模型部署工程师（MLOps） |
| 薪资 | 6 – 9 万元 / 月 · 16 薪（年薪约 96 万 – 144 万 RMB） |
| 公司性质 | 某中型医疗服务公司（猎头代招） |
| 工作地点 | 上海 · 浦东新区 · 上南 |
| 经验 / 学历 | 经验不限 / 本科 |
| 招聘方 | 才多对 · 朱女士（猎头顾问，金牌猎企） |
| 发布时间 | 5 日内新发布 |
| 回复情况 | 今日回复 1 次 |

## 二、职位关键词

`MLOps`　`模型部署`　`CI / CD / CT`

## 三、职位描述

1. **负责 MLOps 端到端平台建设**，搭建训练、评测、发布、推理全流程 **CI / CD / CT 流水线**，制定技术规范。
2. **负责模型部署、灰度发布与弹性伸缩**，优化推理性能，建设模型与系统可观测性及稳定性治理体系。
3. **推进特征 / 数据 / 模型资产管理**，优化计算存储成本，落地安全合规与权限管控，协同多团队推进项目交付。

## 四、职位要求

1. **5 年 + 后端 / 平台经验，3 年 + MLOps / 模型服务化经验**，有平台规模化建设经验优先；
2. **精通 Python / Go / Java 其一**，熟悉 **Kubernetes、DevOps 与分布式训练基础**，具备性能调优与故障排查能力；
3. 理解数据 / 模型版本管理与可观测性体系，**有 TOP / 阿里云 / 百川等相关从业背景**，熟悉 MLOps 主流平台优先。

## 五、特别说明

- 虽然写"经验不限"，但**职位要求里实际要求 5 年+ 后端经验、3 年+ MLOps 经验**——属于资深岗位
- 明确提到 **TOP / 阿里云 / 百川** 等背景优先，暗示公司想要"大厂同款平台"搬到医疗场景
- **CI / CD / CT** 中的 **CT = Continuous Training**，是 MLOps 标准术语（相较传统 CI / CD 多了持续训练环节）
- 16 薪在民营企业中属于较慷慨的发放方式
