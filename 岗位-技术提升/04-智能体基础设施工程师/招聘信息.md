# 智能体基础设施工程师（AI Agent Infrastructure Engineer）

## 一、岗位概览

| 项目 | 内容 |
| --- | --- |
| 职位名称 | 智能体基础实施工程师（AI Agent Infrastructure Engineer） |
| 薪资 | 未明示（对标同级岗位年薪通常 80 – 200 万 RMB） |
| 经验 | 3 年以上后端 / 基础设施开发经验 |
| 学历 | 计算机科学或相关领域本科及以上 |

## 二、岗位概述

作为 **AI Agent 基础设施工程师**，你将负责构建和运维支撑 Agent 运行的**全部底层系统**——包括：

- 安全沙箱环境
- 容器编排
- 多租户隔离
- 云端虚拟机管理
- 任务调度
- 可观测性体系
- 成本优化

你的工作将直接决定 Agent：
- **"能跑多稳"**
- **"能服务多少用户"**
- **"成本能压多低"**

## 三、核心职责

1. **设计并构建 Agent 安全沙箱执行环境**：使每个 Agent 会话在隔离的容器 / 虚拟机中安全运行代码、操作浏览器和文件系统，同时防止恶意行为逃逸。
2. **构建多租户基础设施**：实现用户会话隔离、资源配额管理、弹性扩缩容，确保系统在高并发场景下的稳定性和公平性。
3. **设计 Agent 任务调度和状态管理系统**：支持长时间运行的异步任务、断点续传、故障恢复和优雅降级。
4. **搭建完善的可观测性体系**（日志、指标、链路追踪）：实现 Agent 执行全链路的监控、告警和问题定位。
5. **设计 CI / CD 流水线和基础设施即代码（IaC）实践**：确保基础设施变更可追溯、可回滚、可快速迭代。

## 四、任职要求

### 必备条件
- **计算机科学或相关领域本科及以上学历**；3 年以上后端 / 基础设施开发经验，有大规模分布式系统实战经验
- **精通容器化技术（Docker）和容器编排（Kubernetes）**，有生产环境 K8s 集群管理经验
- **熟悉主流云平台**（AWS / GCP / 阿里云至少一种），了解 VPC、安全组、IAM 等网络和安全配置
- **掌握至少一种主流编程语言**（Go / Python / Rust），有系统级编程能力
- **了解 Linux 系统原理**，包括进程管理、namespace / cgroup、文件系统和网络栈

### 加分项
- 有 **AI / ML 推理服务基础设施经验**（vLLM、TGI、Triton 等）
- 有 **gVisor、Firecracker** 或其他轻量级虚拟化 / 沙箱技术经验
- 有**大规模 WebSocket / 长连接服务的运维经验**
- 熟悉 **Terraform / Pulumi 等 IaC 工具**
- 有 **FinOps 经验**，能系统性优化云资源成本
- **参与过开源基础设施项目**
