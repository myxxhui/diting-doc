## 核心概念

**设计哲学**：Neuro-Symbolic MoE（神经符号混合专家架构；MoE = Mixture of Experts，混合专家）

**核心公式**：
$$
Alpha = (Quant_{Signal} \cap Router(Experts)) \times Kelly_{Position}
$$

**不可能三角（The Impossible Triangle）**：
- **认知边界 (Certainty)**：胜率 ≥ 85%
- **复利增长 (Growth)**：年化复利 ≥ 45%
- **生存底线 (Survival)**：回撤 < 12%
