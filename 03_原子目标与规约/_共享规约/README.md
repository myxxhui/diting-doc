## 二、引用规则

- **本目录文档** = 平台基础规约；不针对单一 pillar
- **各 pillar 独有的规则** → 写入对应 pillar 子目录（如 [03_/极寒防御/](../极寒防御/)）
- **跨 pillar 共享的元数据** → 写入 [`_System_DNA/core_modules/`](../_System_DNA/core_modules/)
- **平台底座技术栈选型** → 写入 [`_System_DNA/global_const.yaml#tech_stack`](../_System_DNA/global_const.yaml) + L4 [04_/共享平台基础](../../04_阶段规划与实践/共享平台基础/)
