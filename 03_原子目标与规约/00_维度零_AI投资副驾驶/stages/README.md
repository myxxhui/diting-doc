## 二、维度零的实践锚定

- **承接 L2 五维度的所有后端能力**（消费方）
- **不做后端引擎判定**，只做"产品体验 + 主动通道 + 价值反馈"
- **6 个 L3 维度中唯一以"用户感知"为成功指标的模块**
- **跨维 step / Mock 退场对齐**：[14_六维度启动期统一节奏表](../../_共享规约/14_六维度启动期统一节奏表.md#14-mock-exit-gates)

## 三、阶段切换准入

| 进阶 | 准入硬条件 |
|---|---|
| stage_1 → stage_2 | 4 product modules 全部上线 + 月度价值证明 PDF 已用 2 个月 + 紧急告警 0 漏推 |
