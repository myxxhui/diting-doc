## 三、4 子模块交付物

| 子模块 | 描述 | 验收指标 |
|---|---|---|
| **持仓体检报告** | Web 首屏 4 色卡片 + 单持仓详情 | 首屏 < 1s；4 色一致 |
| **推荐池与 thesis 卡** | 推荐页 + 5 必填元素 + 3 操作 + PDF | 5 必填 100%；池内 ≤ 5 |
| **紧急告警系统** | 4 红 + 2 橙 + 3 通道 | 5 分钟到达率 ≥ 99.5%；每日红色 ≤ 3 |
| **价值账本** | SCS + EV + 8 象限归因 + 月报 | 月报准时率 100%；SCS ≥ 60 |
