## 二、利润截留扫描仪数据

### 2.1 必需字段

```python
@dataclass
class ProfitCaptureData:
    # 毛利率
    gross_margin: float
    gross_margin_qoq: float
    gross_margin_yoy: float
    
    # 成本与收入增速
    revenue_growth_yoy: float
    cost_growth_yoy: float
    
    # 净利润增速
    net_profit_growth_yoy: float
    
    # 周转率
    receivable_turnover: float
    receivable_turnover_qoq: float
    inventory_turnover: float
    inventory_turnover_qoq: float
```

### 2.2 采集脚本

```python
# deep_strike/data/crawlers/financial_data.py

import akshare as ak

async def fetch_profit_capture_data(symbol: str) -> ProfitCaptureData:
    # 获取财务指标
    df = ak.stock_financial_analysis_indicator(symbol=symbol)
    
    latest = df.iloc[-1]
    prev = df.iloc[-2] if len(df) > 1 else None
    
    return ProfitCaptureData(
        gross_margin=latest["销售毛利率(%)"] / 100,
        gross_margin_qoq=calculate_qoq(latest, prev, "销售毛利率(%)"),
        revenue_growth_yoy=latest["营业收入增长率(%)"] / 100,
        cost_growth_yoy=latest["营业成本增长率(%)"] / 100,
        net_profit_growth_yoy=latest["净利润增长率(%)"] / 100,
        # ...
    )
```

## 四、数据质量

### 4.1 校验规则

```python
def validate_profit_capture_data(data: ProfitCaptureData) -> list[str]:
    issues = []
    
    if data.gross_margin < 0 or data.gross_margin > 1:
        issues.append("毛利率超出合理范围")
    
    if abs(data.gross_margin_qoq) > 0.3:
        issues.append("毛利率环比变化异常大")
    
    return issues
```

### 4.2 缺失处理

| 字段缺失 | 处理方式 |
|---|---|
| 当季财务数据 | 使用上季数据 + 标注 |
| 行业对比数据 | 降级为个股分析 |
| 公告文本 | 仅使用财务数据 |
