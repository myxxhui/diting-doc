## 五、评测

### 5.1 5 必填完整率

```python
def evaluate_thesis_completeness(thesis: dict) -> float:
    required = ["thesis_summary", "evidence_chain", "risks", "valuation_anchor", "action"]
    present = sum(1 for k in required if thesis.get(k))
    return present / len(required)
```

### 5.2 与人工一致率

```python
def evaluate_human_consistency(predictions: list, labels: list) -> float:
    matches = sum(1 for p, l in zip(predictions, labels) if p == l)
    return matches / len(labels)
```
