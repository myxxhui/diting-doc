## 一句话定位

**项目的 Alpha 引擎——把高维碎片信息压缩为有解释、有证据、有失败兜底的研究结论。**

## 核心交付物

- 研究卡片（结论 + 证据链 + 适用周期）
- 研究候选清单（优先级 / 置信度 / 状态机模板建议）
- 议会会议记录（多 Agent 投票、分歧与共识）
- 预期差度量值
