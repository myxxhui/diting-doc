# B 轨说明：共识与逻辑证伪属 C 轨

> [!NOTE] **B 轨（中线）** 不使用「共识落库」「逻辑证伪」「大周期反转」；上述能力属于 **C 轨（长线）**。

- **B 轨**：中线捕手轨；止盈与止损为**中线止损**（如 5%～8% 或 ATR）与**阶段止盈**（如 30%/50%/100% 或阶段结束）；详见 [00_B轨整体设计](00_B轨整体设计.md)、[01_B轨系统设计](01_B轨系统设计.md)。
- **C 轨**：长期捕手轨；使用共识 schema、逻辑证伪、大周期反转；详见 [03_/C轨/03_C轨共识与逻辑证伪规约](../C轨/03_C轨共识与逻辑证伪规约.md)。
