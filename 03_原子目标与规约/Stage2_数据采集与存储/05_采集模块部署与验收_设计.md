<a id="design-stage2-05-goal"></a>
## 本步目标

采集模块部署与验收；K3s 内采集 Job 可运行；真实数据可写入 L1/L2 且可被 Module A/B/C 消费。

<a id="design-stage2-05-points"></a>
## 设计要点

- **验收**：make ingest-test 或 DNA 约定命令退出码 0
- **写入可验证**：至少一种数据源可写入 L1，MarketDataFeed 可读取

<a id="design-stage2-05-exit"></a>
## 准出

采集模块部署成功；写入与读取验证通过；Stage2 准出。
