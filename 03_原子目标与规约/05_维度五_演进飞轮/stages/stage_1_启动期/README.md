| 失败场景 | 兜底策略 |
|---|---|
| Teacher LLM API 调用超额 | 切换备用模型（GPT-4 / Qwen）；蒸馏暂停 |
| Kappa < 0.70 | 增加标注指南培训 + 争议样本仲裁 |

| B 象限误入训练数据 | 立即剔除 + 入审计 + 重新训练 |
