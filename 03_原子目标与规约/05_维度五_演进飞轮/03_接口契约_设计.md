## 五、错误码

| 错误码 | HTTP | 含义 |
|--------|------|------|
| `SE.IDEMPOTENCY_CONFLICT` | 200（带 dedup） | 命中 idempotency，返回首次结果 |
| `SE.GATE_REJECTED` | 403 | 极寒防御门禁拒绝 |
| `SE.EVAL_NOT_PASSED` | 422 | 上线前未通过评测 |
| `SE.HOLDOUT_GATING_BLOCKED` | 422 | **新增**：6 守门指标任一退化 > 5%，被 Holdout 守门阻断 |
| `SE.VERSION_NOT_FOUND` | 404 | 版本不存在 |
| `SE.ROLLBACK_NO_PRIOR` | 409 | 无可回滚版本 |
| `SE.RAG_DEGRADED` | 200（带 partial） | 检索降级 |
| `SE.KAPPA_TOO_LOW` | 422 | **新增**：Kappa < 0.4，训练被阻断 |
| `SE.AUTOPILOT_LIMIT_EXCEEDED` | 403 | **新增**：自动驾驶单笔/日累计超限 |
| `SE.AUTOPILOT_DISABLED` | 403 | **新增**：用户未开启自动驾驶 |
| `SE.AUTOPILOT_USER_REJECTED` | 200 | **新增**：用户拒绝草案 |
| `SE.TEACHER_LLM_QUOTA_EXCEEDED` | 429 | **新增**：Teacher LLM 月度成本上限 |
| `SE.GPU_QUOTA_EXCEEDED` | 429 | **新增**：GPU 训练时长月度上限 |
| `SE.TRAINING_BLOCKED_BY_KAPPA` | 422 | **新增**：训练触发但 Kappa 太低被阻断 |

## 六、版本兼容

同 [极寒防御 §四 版本兼容策略](../极寒防御/03_接口契约_设计.md)。

**Stream schema 版本兼容**：
- 所有 Stream 事件带 `schema_version` 字段
- 新增字段必须可选；删除字段须经过 1 个版本 deprecation 期
- 消费者按 schema_version 选择处理逻辑；未知 schema_version 写入 dlq
