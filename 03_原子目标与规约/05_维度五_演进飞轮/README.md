## 一句话定位

**把"运行结果"变成"下一轮训练数据 + 知识资产 + 个人能力成长"，让人 + 系统在每个周期都能可衡量地变得更强。**

> **承接 L1 双目标的"个人 AI 成长"目标**——这是本项目独有、与"自动炒股器"思路根本不同的产品价值。

## 核心交付物

- 评测报告（按议程 / 周期 / 模块切片）
- 知识库条目（结构化、可检索、有版本）
- 模型 / 策略新版本 + 灰度发布计划
- 个人成长仪表板更新
- 反馈数据集（可被纵深进攻直接消费）
- 外部动作记录（通知 / 工单 / Webhook，**前置必经极寒防御**）

## 后端服务子模块（计划，第 2 批展开）

| 子模块 | 英文键 | 主责 |
|--------|--------|------|
| 评测与回放服务 | `eval_replay_service` | 决策 / 状态迁移 / 外部动作的回放与打分 |
| 知识库服务 | `knowledge_base_service` | 结构化知识 + RAG 索引 + 编辑流 |
| 反馈采集器 | `feedback_collector` | 人审 / 标注 / 事后真相 / 用户点赞 |
| 模型 / 策略注册中心 | `model_registry` | 版本化、灰度、回滚（适配 MLflow / KServe / BentoML） |
| 个人成长仪表板服务 | `growth_dashboard_service` | 把项目工作映射为四岗位能力成长指标 |
| 外部动作边界 | `external_action_boundary` | 通知 / 工单 / Webhook / 第三方 API；**前置必经极寒防御** |
