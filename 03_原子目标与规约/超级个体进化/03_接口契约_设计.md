## 一、契约范围

| 契约 | 用途 |
|------|------|
| `EvalDataset` / `EvalTask` / `EvalReport` | 评测中心 |
| `UserFeedback` / `ImplicitSignal` | 反馈采集 |
| `KnowledgeEntry` / `RAGQuery` / `RAGResult` | 知识库 |
| `RetrospectiveReport` | 复盘 |
| `VersionEntry` / `RolloutPlan` | 版本管理 |
| `ExternalActionRequest` / `ExternalActionResult` | 外部动作边界 |
| `UserProfile` | 用户成长档案 |

## 三、HTTP / gRPC 端点

### 3.1 评测中心
| 端点 | 方法 | 说明 |
|------|------|------|
| `POST /v1/eval/datasets` | gRPC + REST | 注册评测集 |
| `GET  /v1/eval/datasets` | REST | 列出 |
| `POST /v1/eval/tasks` | gRPC + REST | 创建评测任务 |
| `GET  /v1/eval/tasks/{id}` | REST | 任务详情 |
| `GET  /v1/eval/reports/{id}` | REST | 报告详情 |
| `POST /v1/eval/reports/{id}/approve` | REST | 审批 |

### 3.2 反馈
| `POST /v1/feedback/explicit` | REST | 显式反馈 |
| `POST /v1/feedback/implicit` | REST | 隐式信号（埋点） |

### 3.3 知识库
| `POST /v1/knowledge/entries` | gRPC + REST | 写入知识 |
| `GET  /v1/knowledge/entries/{id}` | REST | 单条详情 |
| `POST /v1/knowledge/rag` | gRPC + REST | RAG 检索 |
| `POST /v1/knowledge/entries/{id}/feedback` | REST | 用户对条目反馈 |

### 3.4 复盘
| `POST /v1/retro/run` | gRPC + REST | 触发复盘 |
| `GET  /v1/retro/reports/{id}` | REST | 报告 |
| `GET  /v1/retro/reports` | REST | 列出 |

### 3.5 版本管理
| `POST /v1/versions` | gRPC + REST | 注册版本 |
| `POST /v1/versions/{type}/{id}/{ver}/rollout` | REST | 上线 / 推进 rollout |
| `POST /v1/versions/{type}/{id}/{ver}/rollback` | REST | 一键回滚 |
| `GET  /v1/versions/{type}/{id}` | REST | 列出某 artifact 的全部版本 |

### 3.6 外部动作边界
| `POST /v1/external-actions` | gRPC + REST | 提交动作请求（同步等结果或异步） |
| `GET  /v1/external-actions/{request_id}` | REST | 查询结果 |

### 3.7 用户成长档案
| `GET  /v1/profiles/{user_id}` | REST | 查询（仅本人或授权） |
| `PUT  /v1/profiles/{user_id}` | REST | 编辑 |
| `POST /v1/profiles/{user_id}/export` | REST | 导出（GDPR） |
| `DELETE /v1/profiles/{user_id}` | REST | 删除（GDPR） |

## 四、错误码

| 错误码 | HTTP | 含义 |
|--------|------|------|
| `SE.IDEMPOTENCY_CONFLICT` | 200（带 dedup） | 命中 idempotency，返回首次结果 |
| `SE.GATE_REJECTED` | 403 | 极寒防御门禁拒绝 |
| `SE.EVAL_NOT_PASSED` | 422 | 上线前未通过评测 |
| `SE.VERSION_NOT_FOUND` | 404 | 版本不存在 |
| `SE.ROLLBACK_NO_PRIOR` | 409 | 无可回滚版本 |
| `SE.RAG_DEGRADED` | 200（带 partial） | 检索降级 |
