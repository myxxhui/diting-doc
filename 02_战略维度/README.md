## 这一层负责什么

L2 负责把双目标系统拆成可执行的维度：

- 产品上到底做什么
- 技术上如何支撑它
- 成长上如何映射到岗位能力
- 节奏上如何在 8 个月内推进

## 使用原则

1. 先看产品价值，再看技术复杂度。
2. 先看主线岗位 1/4，再看岗位 3/2 的外溢能力。
3. 先做对当前产品有帮助的设计，再讨论未来平台化。
