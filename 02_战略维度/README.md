## 维度文档模板

每个维度文档应包含：

1. **维度定义**：该维度覆盖什么领域
2. **关键目标**：该维度要达成什么目标
3. **覆盖范围**：该维度包含哪些能力或模块
4. **约束条件**：该维度的边界与限制
