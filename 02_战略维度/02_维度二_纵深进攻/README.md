## 四、协作约定

- **本维度的输入是"过了维度一安检的标的池"**，出维度一才能进维度二
- **每个剧本必须输出"thesis 卡片"**：包含投资逻辑 + SLI 探针清单 + 触发卖出条件
- **本维度不直接持仓**，持仓决策需要架构师手动确认进入维度三的状态机
- **所有剧本输出必须经过维度一的二次校验**（防 LLM 幻觉骗自己）
