## 一、战略目标

不试图覆盖 A/H 全市场所有机会。**只在自己有信息+认知优势的少数赛道做"纵深进攻"**——核心思路：

> "Leverage 在能力圈，而不是 leverage 在杠杆。"

**核心使命**：在能力圈内深挖，使每一笔投资都建立在**可验证的、机器可读的逻辑链**上，而非"听说了 / 跟风了"。

## 二、剧本驱动的方法论

每一个"剧本"对应一类**重复出现的、可被研究的"非对称机会"**：
- 不是"个股推荐"，而是"识别这类机会的方法论"
- 输出"thesis 卡片"——这一笔投资的核心论点 + 可观察的 SLI 探针 + 触发卖出条件
- thesis 卡片必须能被维度三/四消费（即"投资逻辑也是被监控的对象"）

## 三、决策机制

| 决策类型 | 判定 | 动作 |
|---|---|---|
| `propose` | 剧本命中，且证据强度 ≥ 0.7 | 输出 thesis 卡片，进入"待人工确认池" |
| `watch` | 剧本部分命中，证据强度 0.4–0.7 | 进入"观察池"，触发新一轮数据采集 |
| `discard` | 剧本未命中或证据 < 0.4 | 写入"已研究但未持仓"日志（用于后续训练负样本） |

**不允许 AI 自动建仓**：thesis 卡片必须经架构师"建仓确认"按钮人工确认才能进维度三。

## 四、关键能力清单（10 剧本概要）

1. **利润截留扫描仪**：识别"成本下降快于收入下降 / 毛利率拐点 / 经营杠杆释放"的早期信号
2. **S 曲线渗透率**：监控新技术/新产品在产业链中的渗透率拐点（如新能源车、AI 算力）
3. **产业链瓶颈嗅探**：识别产业链中"卡脖子"环节的稀缺性溢价（如稀土永磁、半导体设备）
4. **产能出清**：识别"周期底部产能出清后龙头集中度提升"的机会（如生猪、光伏硅料）
5. **国产替代攻坚**：识别"被海外卡脖子"且"国内有突破"的赛道
6. **出海/全球化**：识别"中国制造/中国服务模式输出海外"的机会
7. **中特估估值重塑**：识别"低估值高分红的国央企"的价值重估机会
8. **政策驱动主升浪**：识别"政策密集出台后行业 EPS 与估值双升"的机会
9. **困境反转个股**：识别"业绩在底部反转拐点"的个股
10. **细分龙头扩品类**：识别"在主业稳固后扩展第二增长曲线"的机会

## 五、能力边界（不做什么）

| 边界 | 不做的事 | 原因 |
|---|---|---|
| 短线交易信号 | 不做日内/周内交易策略 | 与 L1 的"超长期复利"目标冲突 |
| 量价信号 | 不基于 K 线/MACD/KDJ 等技术指标做决策 | 不在能力圈内 |
| 跟风炒作 | 不追任何"概念热点"（如纯 AI 概念、纯人形机器人概念） | 缺乏可验证 SLI |
| 替代基本面研究 | 不替代深度行业研究，仅做"机会筛选 + 证据组织" | 必须人工 review |
| 自动建仓 | 不直接产生交易订单 | 永久禁止 |

## 六、L3 子模块映射

| L3 子模块 | 职责 |
|---|---|
| `playbook_engine` | 剧本工作流引擎（基于 LangGraph） |
| `evidence_chain_builder` | 证据链构建器（结构化抓取财报/公告/产业数据） |
| `thesis_card_generator` | thesis 卡片生成器 |
| `confidence_scorer` | 置信度评分器 |
| `sandbox_runner` | gVisor/Firecracker 沙箱（执行 Agent 生成的代码） |
| `human_confirmation_gate` | 人工确认门禁（前端"建仓确认"） |

## 七、前端接入

- **剧本驾驶舱**：当前各剧本扫到的标的池、置信度排名
- **thesis 卡片审阅页**：架构师 review + 建仓确认入口
- **能力圈管理面板**：架构师维护"我懂的赛道清单"，剧本只在这些赛道上跑
