> - **顶层概念**: [A股分析追踪平台目标与边界](../../01_顶层概念/04_A股分析追踪平台目标与边界.md)

## 主业务链路

```mermaid
flowchart LR
  source[数据源]
  ingest[采集与清洗]
  knowledge[知识与检索]
  agents[Agent工作流]
  output[研究卡片与thesis卡片]
  track[持续跟踪与提醒]
  feedback[人工反馈与复盘]

  source --> ingest --> knowledge --> agents --> output --> track --> feedback
  feedback --> knowledge
```

## 工程链路

```mermaid
flowchart LR
  product[产品需求]
  workflow[Prompt与Workflow]
  eval[评测集与回归]
  deploy[灰度发布]
  observe[观测与告警]
  improve[反馈与迭代]

  product --> workflow --> eval --> deploy --> observe --> improve --> workflow
```

## 链路能力映射

- `知识与检索`、`Agent 工作流`：研究编排能力
- `灰度发布`、`回归`：评测与发布能力
- `观测与告警`、`推理底座`：基础设施治理能力
- `Runtime / 沙箱 / 长任务`：运行时与隔离能力
