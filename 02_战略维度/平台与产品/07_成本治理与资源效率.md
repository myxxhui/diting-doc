## 目标

成本治理的目标，不是压缩一切投入，而是让副业长期可持续，并把资源投入集中在真正有价值的环节。

## 三类成本

### 1. 推理成本
- 商业 API 成本
- 自托管 GPU / CPU 成本
- token 消耗与上下文膨胀

### 2. Runtime 成本
- 长任务驻留
- 沙箱冷启动
- 多会话资源占用

### 3. 存储与观测成本
- 原始资料归档
- 向量索引
- 日志、指标、追踪

## 主要策略

- 模型分级：简单任务用轻模型，关键任务用重模型
- 缓存优先：检索、摘要、结构化结果尽可能缓存
- 异步优先：长任务避免阻塞在线路径
- 分层存储：热数据、冷数据、评测数据分离
- 按阶段投入：没有产品价值就不要过早上昂贵基础设施

## 能力落点

- FinOps：资源治理与推理成本优化
- Runtime：回收策略、冷启动与任务密度优化
- 发布治理：灰度、回滚与无效试错成本控制
