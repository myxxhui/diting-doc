## 五、协作约定

- **本维度是全局熔断器，不是某个串行步骤**：任何其他维度的对外输出都必须先过本维度的决策门禁
- **一票否决制**：命中任意防御红线 → 写入永久黑名单；解除需要架构师手动覆盖
- **本维度不产生研究结论**（属于维度二）；不维护状态机生命周期（属于维度三/四）；不直接执行外部动作（属于维度五的"外部动作边界"）
